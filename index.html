Nihongo Shooter

Speed run learning a language. I made this game because animations in Duolingo slowed me down. You can learn Spanish, French, German, Italian, Japanese, Korean, Portuguese, Russian, Chinese, Hebrew, or you can learn English if you know one of those languages. See source on github. Note you can use the arrow and 1, 2, 3 keys to shoot with a keyboard.

If you'd like updates on new developments of mine, follow me on twitter or substack.
